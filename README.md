# Two Line Element Set Converter

Dependencies:
   1. pyephem
   2. print_function
   3. re

Run: 
   1. open terminal
   2. clone this repo 
   3. cd to TLE-Converter
   4. pip install -r requirements.txt
   5. python3 tle-converter.py

# Usage : 
Built for tracking things in earths orbit.
